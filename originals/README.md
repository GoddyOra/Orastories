Place your canonical book files (PDF/EPUB/TXT) in this folder before running the upload script.

Files will be uploaded to the configured S3 bucket under the `originals/` prefix by default.

Supported extensions: `.pdf`, `.epub`, `.txt`, `.md`, `.jpg`, `.jpeg`, `.png`

Do NOT commit large binaries into the repo unless you intentionally want them in Git history. Use this folder as a local staging area only.
